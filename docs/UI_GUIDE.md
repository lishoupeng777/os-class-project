## 🎯 功能交互

### 1️⃣ 仪表板 (Dashboard)
**位置**: 左侧菜单 → "📊 仪表板"

**显示内容**:
- 💾 **磁盘使用情况** - 总容量 256MB，已用百分比进度条
- 🔗 **Inode 节点** - 总 1024 节点，使用率统计
- 📂 **文件列表** - 当前目录所有文件和文件夹
- 👤 **当前用户** - 显示已登录用户

**操作**:
```
✓ 单击文件项 → 选中文件（高亮显示）
✓ 双击文件夹 → 进入该目录（如果实现导航）
✓ 右键 → 快捷菜单（复制、删除、属性等）
```

### 2️⃣ 用户登录 (Login)
**触发方式**: 左侧菜单 → "🔐 用户登录"

**弹窗内容**:
```
┌───────────────────────────┐
│ 🔐 用户登录              │
├───────────────────────────┤
│ 用户名: [_____________]  │
│ 密码:   [_____________]  │
│                          │
│     [取消]    [登录]     │
└───────────────────────────┘
```

**操作流程**:
1. 输入用户名（如 `root`, `alice`, `bob`）
2. 输入密码
3. 点击 "登录" 按钮
4. 成功后左侧显示新用户名

### 3️⃣ 新建文件夹 (Mkdir)
**触发方式**: 左侧菜单 → "📂 新建文件夹"

**弹窗内容**:
```
┌───────────────────────────┐
│ 📂 新建文件夹           │
├───────────────────────────┤
│ 文件夹名称: [_________]  │
│                          │
│     [取消]    [创建]     │
└───────────────────────────┘
```

**操作流程**:
1. 输入文件夹名称
2. 点击 "创建" 按钮
3. 成功后文件列表刷新，新文件夹出现
