                        核心源代码文件（4个）
════════════════════════════════════════════════════════════════════════════════

✅ main.c (150行)
   ├─ 用途: 程序入口 + 命令行界面
   ├─ 功能: 
   │  ├─ 显示帮助菜单
   │  ├─ 解析用户输入的命令
   │  └─ 路由到对应的命令处理函数
   └─ 例子: "mkdir dir1" → cmd_mkdir("dir1")

✅ vfs.c (500+行) ⭐ 核心文件
   ├─ 用途: 文件系统核心算法实现
   ├─ 关键函数:
   │  ├─ iget/iput - Inode 缓存管理 (O(1)查找)
   │  ├─ balloc/bfree - 块分配/释放
   │  ├─ ialloc/ifree - Inode 分配/释放
   │  ├─ namei - 路径解析
   │  ├─ access - 权限检查
   │  └─ format_disk - 磁盘初始化
   └─ 新增: 隐患修复代码

✅ vfs.h (170行) ⭐ 核心文件
   ├─ 用途: 数据结构定义 + 系统常数
   ├─ 定义的结构体:
   │  ├─ super_block - 超级块（磁盘元数据）
   │  ├─ dinode - 磁盘Inode（32字节）
   │  ├─ minode - 内存Inode（带缓存管理）
   │  ├─ dir_entry - 目录项（16字节）
   │  └─ user - 用户信息
   └─ 常数: BLOCKSIZ=512, USERNUM=10, NOFILE=20等

✅ commands.c (1350+行) ⭐ 核心文件
   ├─ 用途: 25个命令的实现
   ├─ 用户命令 (11个):
   │  ├─ login, logout, whoami - 用户管理
   │  ├─ mkdir, chdir, dir, pwd - 目录管理
   │  ├─ create, open, close, delete - 文件操作
   │  └─ read, write - 文件I/O
   ├─ 权限命令 (4个):
   │  ├─ chmod - 修改权限
   │  ├─ chown - 修改所有者
   │  ├─ useradd, userdel - 用户管理
   │  └─ passwd - 改密码
   ├─ 文件操作 (6个):
   │  ├─ copy - 复制文件
   │  ├─ link - 硬链接
   │  ├─ symlink - 软链接
   │  ├─ rename - 重命名
   │  ├─ touch - 快速创建
   │  ├─ cat - 查看内容
   │  └─ stat - 文件信息
   └─ 新增: 隐患#5修复代码

════════════════════════════════════════════════════════════════════════════════
                      虚拟磁盘文件（1个）
════════════════════════════════════════════════════════════════════════════════

✅ virtual_disk.bin (4MB)
   ├─ 用途: 虚拟磁盘存储
   ├─ 结构:
   │  ├─ Block 0: 引导块（启动信息）
   │  ├─ Block 1: 超级块（文件系统元数据）
   │  ├─ Block 2-33: Inode表（32块，共128个Inode）
   │  └─ Block 34-545: 数据区（512块存放文件数据）
   ├─ 特性:
   │  ├─ 持久化存储（程序启动时加载，退出时保存）
   │  ├─ 4MB 总容量（512字节/块 × 8192块）
   │  └─ 可存储约 68KB 的最大文件
   └─ 位置: d:\trae project\os\virtual_disk.bin
